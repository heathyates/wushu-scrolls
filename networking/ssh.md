## Config 

This file allows aliases to connect to a external server. 

```
Host krelln
    HostName 192.168.1.2
    User hlyates
    Port 22
    IdentityFile ~/.ssh/id_ed25519

Host arvon
    HostName 192.168.1.3
    User hlyates
    Port 22
    IdentityFile ~/.ssh/id_ed25519

```
